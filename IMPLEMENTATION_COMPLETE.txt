
### Step 2️⃣: Setup & Run

**Terminal 1 - Backend:**
```powershell
cd c:\MSCIT SEM4\Project\ResumeAINew\backend
python -m venv venv
.\venv\Scripts\Activate.ps1
pip install -r requirements.txt
Copy-Item .env.example .env

python app.py
```

**Terminal 2 - Frontend (new terminal):**
```powershell

npm run dev
```

```

Upload sample files

```

## 🎯 What Each Application Does

### Backend (Python)
- Runs on http://localhost:5000
- Receives uploaded files
- Extracts text using PyPDF2
- Calls OpenAI API for analysis
- Returns structured JSON data

### Frontend (React)
- Runs on http://localhost:3000
- Shows beautiful upload interface
- Displays analysis results
- Provides AI chatbot
- Communicates with backend

### OpenAI API
- Analyzes resume vs job description
- Powers the chatbot
- Generates explanations
- Requires API key in .env

## 💡 Tips for Success

1. **API Key**: Must be valid OpenAI API key (starts with sk-)
2. **Files**: Use PDF or TXT format (< 16MB each)
3. **Sample Files**: SAMPLE_RESUME.txt and SAMPLE_JOB_DESCRIPTION.txt included
4. **Terminals**: Keep both backend and frontend running
5. **Errors**: Check console for detailed error messages

## 🐛 Common Issues & Solutions

| Problem | Solution |
|---------|----------|
| "Module not found" | Run `pip install -r requirements.txt` |
| "npm ERR!" | Run `npm install` in frontend/ |
| "Port 5000 in use" | Kill other processes or use different port |
| "OpenAI error" | Check .env has valid API key |
| "Files won't upload" | Check file size < 16MB and format PDF/TXT |

See QUICK_REFERENCE.md for more solutions.
